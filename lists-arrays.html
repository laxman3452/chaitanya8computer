3. Reviewing the “Numbers” Program

Numbers = [6, 49, 33, 8, 72, 1060, 198] counter = 0 while counter < len(Numbers): print(Numbers[counter]) counter += 1

Explanation

  • The counter starts from 0.
  • The loop moves through each index position.
  • counter += 1 increases the counter by 1.
  • len(Numbers) gives the total number of elements.

4. Python List Methods

Method Description Example
append() Adds an item at the end numbers.append(50)
insert() Inserts item at specific position numbers.insert(0, 100)
sort() Sorts list in ascending order numbers.sort()
reverse() Reverses current order numbers.reverse()
remove() Removes specific item numbers.remove(10)
pop() Removes last item numbers.pop()
len() Returns total elements len(numbers)
