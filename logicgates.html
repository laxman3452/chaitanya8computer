Boolean Logic & Logic Gates

Subject: Computer Science (Introductory) | Duration: 45 Minutes

Learning Objectives

  • Define the three main Boolean operators: AND, OR, and NOT.
  • Evaluate Boolean expressions as True (1) or False (0).
  • Identify and draw standard symbols for Logic Gates.

1. The Language of Computers

Discussion: "How does your phone know to unlock only when both Face ID is correct AND you swipe up?"

Computers make decisions using Boolean Logic. Every operation is based on just two values:

  • 1 = True
  • 0 = False

2. The Core Operators

A. AND Operator (The Strict Operator)

Rule: Both conditions must be TRUE.

Example: To be a Samsung Phone, it must: (Be a phone) AND (Be made by Samsung).

Result: 0

B. OR Operator (The Generous Operator)

Rule: At least one condition must be TRUE.

Example: A device is TRUE if it is: (A phone) OR (Made by Samsung).

Result: 0

C. NOT Operator (The Inverter)

Rule: It reverses the result.

Example: NOT Samsung means any brand except Samsung.

Result: 1

3. Activity: Human Logic Gates

Scenario A (AND)

"Stand up if you are wearing a watch AND white shoes."

Scenario B (OR)

"Stand up if you are wearing a watch OR white shoes."

Scenario C (NOT)

"Stand up if you are NOT wearing glasses."
